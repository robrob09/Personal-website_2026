# Personal Portfolio Website

Статический двуязычный сайт-портфолио с двумя версиями подачи контента: **IT** и **STEM**. Проект построен без фреймворков и сборщика: страницы написаны на HTML, стили — на CSS, интерактивность реализована на vanilla JavaScript, а контент подгружается из JSON.

Сайт включает главную страницу-портфолио, отдельные страницы с резюме и страницу career path. Для пользователя доступны переключение языка, темы и трека, а состояние интерфейса сохраняется между переходами.

## Что есть в проекте

- **Два трека контента:** IT и STEM.
- **Два языка:** English и Русский.
- **Светлая и тёмная темы** с локальным хранением выбранного режима.
- **Data-driven архитектура:** основное содержимое хранится в `source/data/*.json`.
- **Отдельные страницы резюме** с веб-рендером одностраничного CV и кнопкой скачивания PDF.
- **Страница career path** для расширенного представления опыта и направлений развития.
- **Локальные шрифты и ассеты**, без зависимости от внешних CDN.
- **Аккуратная клиентская навигация:** сохранение позиции при переключении языка и трека, фиксированный header, прокрутка по секциям.

## Структура

```text
.
├── index.html / stem.html            # главные страницы EN
├── career-*.html                     # страницы career path EN
├── resume-*.html                     # страницы resume EN
├── ru/                               # русские версии страниц
└── source/
    ├── css/                          # стили
    ├── js/                           # логика интерфейса и рендеринга
    ├── data/                         # JSON-данные для IT/STEM и EN/RU
    ├── img/                          # иконки, аватары, превью проектов
    ├── fonts/                        # локальные шрифты
    └── latex/                        # исходники LaTeX для резюме
```

## Запуск

Так как контент загружается через `fetch()` из JSON, проект нужно открывать **через локальный сервер**, а не напрямую через `file://`.

Простой вариант:

```bash
python -m http.server 8000
```

После этого откройте в браузере:

- `http://localhost:8000/index.html` — IT track, English
- `http://localhost:8000/stem.html` — STEM track, English
- `http://localhost:8000/ru/index.html` — IT track, Russian
- `http://localhost:8000/ru/stem.html` — STEM track, Russian

Также подойдёт VS Code Live Server или любой другой статический HTTP-сервер.

## Где редактировать контент

- **Тексты, секции, проекты, контакты:** `source/data/it-en.json`, `it-ru.json`, `stem-en.json`, `stem-ru.json`
- **Изображения и иконки:** `source/img/`
- **Превью проектов:** `source/img/projects/`
- **Шрифты:** `source/fonts/`
- **LaTeX-версии резюме:** `source/latex/`
- **Готовые PDF для кнопок скачивания:** `source/pdf/`
